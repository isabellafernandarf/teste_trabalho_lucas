Semantic UI

O Semantic UI é um framework de desenvolvimento web que oferece uma coleção de elementos de interface de usuário (UI) com nomes semanticamente significativos. Ele é projetado para facilitar a criação de sites e aplicativos web com uma aparência e comportamento agradáveis e consistentes.
